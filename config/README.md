# Mappings

## eap-mapping.json

De eap-mapping.json is de oude config file de gebruikt werd voor de publicatie van de specificaties van openbaar domein. Deze omvat alle standaarden in één grote file. 
Bij de herpublicatie naar een toolchain 4 compatibele versie is er voor gekozen om de config file op te splitsen per thema. Deze files moeten gebruikt worden voor herpublicaties of aanpassingen aan de spec. 
De "master"-file dient niet meer gebruikt te worden.
